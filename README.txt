iTunes-Notify
========

.. image:: https://github.com/Roasbeef/iTunes-Notify/raw/master/Demo.png
   :align: left

Installation
============

Installing using pip. ::

    pip install itunesnotify

Using iTunes-Notify
==============

Available commands. ::

    itunes-notify run 
    # start the notification process

    itunes-notify stop 
    # end the notification process
